بهینه سازی منابع

این قانون زمانی فعال می‌­شود که DSR (Digi Seo Report) تشخیص دهد، کوچک کردن فایل‌ها می‌­تواند اندازه منابع را کاهش دهد.

توضیح

کوچک کردن به فرآیند حذف داده­‌های غیرالزامی یا زاید، بدون تأثیر گذاشتن بر نحوه پردازش شدن منابع توسط مرورگر گفته می‌­شود. مثلا حذف کد کامنت‌ها، حذف کدهای استفاده نشده، استفاده از متغیر و نام توابع کوتاه‌تر و غیره. برای اطلاعات بیشتر به مقاله با Minify بیشتر آشنا شوید مراجعه کنید.

توصیه­‌ها

شما باید منابع HTML، CSS و جاوا اسکریپت خود را کوچک کنید.
